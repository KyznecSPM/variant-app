## Техническое описание

- Монорепозиторий с двумя пакетами: `api/` (сервер) и `client/` (UI); общие типы в `shared/`.
- API на Fastify с TypeBox‑схемами: валидация входных данных и явный контракт ответа.
- Генерация текста через OpenAI Chat Completions (модель `gpt-4o-mini`), системный промпт хранится в API‑сервисе.
- Клиент на React + Vite, маршрутизация через React Router.
- Состояние приложения хранится в контексте: отдельные контексты для state и actions, чтобы избежать лишних ререндеров.
- Персистентность через `localStorage` (ключ `coverLetters`), без серверной БД.
- UI стили — CSS Modules + дизайн‑токены в `client/src/styles/tokens.css`.
- Клиент обращается к `POST /api/cover-letter/generate`.
- Запрос/ответ типизированы общими интерфейсами из `shared/types`.
- Ошибки API возвращаются с кодом 500 и сообщением; клиент показывает ошибку в консоли (без UI).

## Запуск

Локально: `npm run dev` стартует одновременно `api` и `client`.
